## 📍 Section 3: Product Area (Center)

### Search Bar
Located at the top of the product area.

**Search By:**
- Product name (e.g., "Speed Blaster")
- Toy number (e.g., "HW-001")
- Series name (e.g., "2024 Series")
- Year (e.g., "2024")

**Keyboard Shortcuts:**
- `Ctrl+F` - Focus search bar
- `/` - Quick focus search
- `Esc` - Clear search
- `X` button - Clear search (mouse)

**Features:**
- Real-time filtering
- No submit button needed
- Case-insensitive
- Searches within selected category

### Product Grid

**Layout:**
- Responsive grid (1-4 columns based on screen size)
- Smooth animations when switching categories
- Hover effects on each card

**Product Card Information:**

Each card shows:
1. **Product Image** - Large emoji representation
2. **Rarity Badge** - Color-coded corner badge
   - 🟢 Green = Common
   - 🔵 Blue = Premium
   - 🟠 Orange = Treasure Hunt
   - 🟡 Gold (pulsing) = Super Treasure Hunt
3. **Product Name** - Bold, clear text
4. **Toy Number** - Code like "HW-001"
5. **Series** - Edition or collection name
6. **Price** - Large, gradient styled
7. **Year** - Release year
8. **Add to Cart Button** - Orange with + icon

**Hover Effects:**
- Card lifts up (3D effect)
- Image spins 360°
- Border glows orange
- Speed lines appear
- Shine sweeps across

**Adding to Cart:**
1. Click "Add to Cart" button
2. Product image flies to cart
3. Particle trail animation plays
4. Hear confirmation beep
5. Cart updates instantly

## 📍 Section 4: Pit Stop Cart (Right)

### Cart Header
- 🛒 Shopping cart icon
- Item count badge
- "Pit Stop" title
- Racing stripe divider

### Cart Items

Each item displays:
- Product image (emoji)
- Product name
- Toy number
- Unit price

**Quantity Controls:**
- `−` button - Decrease quantity
- Number display - Current quantity
- `+` button - Increase quantity
- 🗑️ button - Remove item

**Quantity Behavior:**
- Decreasing to 0 removes item
- Removal triggers slide-out animation
- Hear removal sound effect

### Cart Summary

Located at the bottom of cart:

**Breakdown:**
1. **Subtotal** - Sum before discounts
2. **Fuel Boost** - Discount amount (if applied)
   - Shows ⚡ icon
   - Yellow color
   - Percentage from header
3. **Tax** - 8.5% applied
4. **Grand Total** - Final amount to pay
   - Large racing font
   - Orange gradient
   - Digital display style

**Checkered Flag Divider:**
- Racing pattern separator
- Visual break before checkout

**Finish Line Button:**
- Large yellow gradient button
- 🏁 Racing flag icon
- "FINISH LINE" text
- Disabled if cart empty
- Glows when hovering

## 💳 Checkout Process

### Step 1: Click Finish Line
- Button located at bottom of cart
- Must have items in cart
- Click to open payment modal

### Step 2: Payment Dashboard Opens

**Modal Features:**
- Dark background with blur
- Orange glowing border
- Dashboard-style design
- Total amount displayed

**Payment Methods:**

1. **💵 Cash (Green)**
   - Traditional cash payment
   - Green gradient card

2. **💳 Credit Card (Blue)**
   - Card payment option
   - Blue gradient card

3. **💰 Digital Wallet (Amber)**
   - Apple Pay, Google Pay, etc.
   - Amber gradient card

**How to Select:**
- Click on any payment method card
- Selected card shows checkmark
- Card glows orange when selected
- "Complete Sale" button activates

### Step 3: Complete Sale
- Click "Complete Sale" button
- Disabled until method selected
- Triggers victory animation

### Step 4: Victory Celebration! 🎉

**Amazing effects:**
1. **Confetti Explosion**
   - Racing colors burst
   - Particles fall from top
   - Multiple layers

2. **Victory Lap Car**
   - Race car zooms across screen
   - Spins 360° while moving
   - Speed trail follows

3. **Checkered Flags**
   - Float down from top
   - Rotating animation
   - Multiple flags

4. **Success Message**
   - "Victory Lap!" title
   - "Sale Completed Successfully!"
   - Racing stripes
   - Spring animation

**What Happens:**
- Animation plays for 3.5 seconds
- Success sound plays
- Cart automatically clears
- Discount resets to 0%
- Ready for next sale!

## 💡 Tips & Tricks

### Shopping Efficiency
1. **Use Keyboard Shortcuts**
   - Fastest way to switch categories
   - Press `1-5` for instant switching

2. **Quick Search**
   - Press `/` to jump to search
   - Type partial names
   - Finds across all fields

3. **Bulk Adding**
   - Add to cart first
   - Adjust quantity in cart
   - Easier than multiple adds

### Cart Management
1. **Quick Remove**
   - Click trash icon for instant remove
   - Or decrease to 0 for smooth animation

2. **See Total Live**
   - All prices update instantly
   - No need to calculate
   - Tax included automatically

3. **Apply Discounts Early**
   - Set Fuel Boost percentage before checkout
   - See savings immediately
   - Can change anytime

### Fast Checkout
1. **Keep Payment Method in Mind**
   - Decide before clicking Finish Line
   - One click to select
   - Enter to confirm

2. **Enjoy the Victory**
   - Don't click during animation
   - Let it finish naturally
   - Cart clears automatically

## 🎯 Common Tasks

### Task 1: Sell a Single Item
1. Select category (or search)
2. Hover over product card
3. Click "Add to Cart"
4. Watch animation
5. Click "Finish Line"
6. Select payment method
7. Click "Complete Sale"
8. Enjoy victory animation

**Time:** ~10 seconds

### Task 2: Sell Multiple Items
1. Browse and add items to cart
2. Adjust quantities as needed
3. Apply discount if applicable
4. Review total in cart
5. Click "Finish Line"
6. Complete payment
7. Victory celebration

**Time:** ~30 seconds

### Task 3: Search for Specific Car
1. Press `/` or `Ctrl+F`
2. Type car name or number
3. Find in results
4. Add to cart
5. Press `Esc` to clear search
6. Continue shopping

**Time:** ~5 seconds

### Task 4: Apply Store Discount
1. Look at Fuel Boost in header
2. Click desired percentage (5%, 10%, 15%)
3. See discount apply to cart
4. Continue with checkout

**Time:** 2 seconds

## 🔧 Troubleshooting

### Product Not Showing?
- Check selected category
- Clear search bar (Esc)
- Try different category
- Refresh if needed

### Cart Not Updating?
- Check if product added (animation played)
- Look at cart count in header
- Scroll cart if many items
- Try again if issue persists

### Animation Lag?
- Close other browser tabs
- Use modern browser (Chrome, Firefox, Edge)
- Check system resources
- Disable browser extensions if needed

### Sound Not Playing?
- Check browser allows audio
- May need user interaction first
- Volume set correctly
- Browser permissions enabled
